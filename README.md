# Form Validation Major Project

A simple web-based form validation project built using HTML, CSS, and JavaScript. It validates user inputs such as name, email, and password, providing real-time feedback and error messages to ensure accurate data entry.

## Features

* Real-time validation
* Email format checking
* Password validation
* Error message display
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript
